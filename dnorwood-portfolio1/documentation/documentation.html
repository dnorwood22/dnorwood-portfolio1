About Template

Estro is an HTML template for a virtual card and a blog.

The template is built on the basis of a popular library – Bootstrap. Along with standard scripts, Estro uses Bower package manager for uploading and updating libraries. The scripts are divided into simple parts. You can easily activate or disable any feature.
Estro includes the following sections: welcome section, about me section, resume section, portfolio section, contacts with a working feedback form and a blog.

Visit template's online page

Template Pack

Folders and files

Here is a file structure of the package:

  • documentation – folder with documentation
  • static – folder with content
    • images – folder with all images
    • scripts – folder with custom scripts for template
    • styles – folder with custom CSS and LESS
    • vendor – folder with vendor scripts and plugins
    • video – folder with video files (+ poster image)
  • html files
  • php files

Navigation

Template uses bootstrap navbar. More information you can find here.

Customizing

Navigation has two basic styles: colored and black. By default navbar is colored to main color. For using black style you can only add class .black-style. to <nav>.

<nav class="navbar navbar-default navbar-fixed-top"> required classes for navigation
<nav class="navbar navbar-default navbar-fixed-top black-style"> black style for navigation

When a page is scrolled, a new class .navbar-scrolled is added to the navigation. This class is responsible for colorizing the navigation.

Welcome section

Welcome section has some variants for background:

  • – Static Image
  • – Parallax Image
  • – Image slider
  • – Video (local source)

Fullscreen option

Welcome section can be resized to a fullscreen area. Use class .js-fullscreen for this option.

<section class="welcome-section-wrapper js-fullscreen">
    <div class="welcome-section-inner container">
        ...
    </div>    
</section>
Class .container is needed inside. Without this class script doesn't work.

Static image

Set static image background in less/css. Change 'background' for 'body' class .static-image.

body.static-image {
  ...
  background: url(".../background.jpg") no-repeat center center fixed;
  -webkit-background-size: cover;
  -moz-background-size: cover;
  -o-background-size: cover;
  background-size: cover;
}
or open theme.less: 
@background-static-image: url(".../static_welcome_bg.jpg"); //only for static welcome image bg

Add this class to <body>

<body class="static-image">>
    ...
</body>
Overlay

For using color overlay for static image add class .overlay to <body>.

This static image (with overlay) displays as background in all templates between sections include footer. If you want to use different backgrounds use css to create it.

Parallax image

Connect to an html following scripts and add class '.js-welcome-parallax' to the section '.welcome-wrapper'.

<script src="static/vendor/parallax.js/parallax.min.js"></script>
<!-- parallax main library (parallax.js plugin) -->
<script src="static/scripts/parallax.js"></script>
<!-- init and settings for parallax backgrounds -->

<section class="welcome-wrapper js-welcome-parallax">
    ...
</section>

You can change background image in parallax.js.

// Select background for welcome section
$('.js-welcome-parallax').parallax({
    imageSrc: 'static/images/backgrounds/background.jpg',
    position: 'center top'
});

Image Slider

Sliding images via a jQuery plugin VEGAS. Connect following files for using this feature:

CSS
<link href="static/vendor/vegas/dist/vegas.min.css" rel="stylesheet" media="screen">
<!-- Vegas slideshow styles -->
JS
<script src="static/vendor/vegas/vegas.min.js"></script>
<!-- Vegas background slideshow -->
<script src="static/scripts/vegas.js"></script>
<!-- init and settings for Vegas -->

Use class .js-bg-slides.

<section class="welcome-section-wrapper js-fullscreen js-bg-slides">...</section>

Settings are in the static/scripts/vegas.js. There you can select images for sliding.

Video

Background video implemented via Vide. Connect following scripts for using this feature:

<script src="static/vendor/vide/dist/jquery.vide.min.js"></script>
<!-- background video (vide.js plugin) -->
<script src="static/scripts/videobg.js"></script>
<!-- init and settings for background video -->

Settings are in the static/scripts/videobg.js. There you can select videos.

Use class .js-bg-video

<section class="welcome-section-wrapper js-fullscreen js-bg-video">...</section>

Use class .overlay to body to fill video by a color. It's same overlay for static image.

<div class="background-video js-bg-video overlay"></div>

Image preview

Full-size preview implemented via Fancybox 2 script. For using this plugin you need connect the following files:

CSS
<!-- Fancybox2 styles -->
<link href="static/vendor/fancybox/source/jquery.fancybox.css?v=2.1.5" rel="stylesheet" media="screen" />

JS
<!-- fancybox main library (fancybox 2 plugin) -->
<script src="static/vendor/fancybox/source/jquery.fancybox.pack.js?v=2.1.5"></script>
<!-- mousewheel plugin (fancybox 2 plugin. Optional) -->
<script src="static/vendor/fancybox/lib/jquery.mousewheel-3.0.6.pack.js"></script>
<!-- init and settings for fancybox -->
<script src="static/scripts/fancybox.js"></script>

Google map

Google map implemented via gmaps.js

Connect the following scripts to use this feature:

<script src="https://maps.googleapis.com/maps/api/js"></script>
<script src="static/vendor/gmaps/gmaps.min.js"></script>
<script src="static/scripts/maps.js"></script>

Set point

For setting marker you need edit an html. Find gmap section and edit settings:

<div id="google-map-container" class="google-map-container" data-lat="-00.000000" data-lng="-00.000000"></div>

Drawer

The template has left drawer. It is a block with some information. Drawer appears and hide by click on a button. The button is displayed in the form of a logo.

The button

<div class="js-drawer-toggle drawer-toggle" data-target=".drawer-wrapper">
    <div class="icon-drawer-wrapper">
        place your logo here
    </div>
</div>

The left drawer

<div class="drawer-wrapper hide">
    <div class="scrollable">
        <div class="tilt-effect-wrapper photo-main">
            place your photo here
        </div>
        <div class="drawer-inside text-white">
            place your information here
        </div>
    </div>
</div>

Tilt effect

You can use tilt effect on any photo in this template. By default effect applied on drawer photo only. More information about script you can find here.

If you want use left drawer without tilt effect connect the following file – drawer.js. For using tilt effect in left drawer replace file with drawer-tilt.js

Responsive layout

The template supports responsive/adaptive layout via responsive.css.

Retina support

The template is retina-ready. Connect the following script to enable this feature:

<script src="static/vendor/retinajs/dist/retina.min.js"></script><!-- retina main library (retina.js plugin) -->

Themes and colors

Template already has some themes. For easily customizing theme edit theme.less and @import it to the main.less.

Typical theme.less file:

// USER SETTINGS (Theme)
//----------------------------------------------------

// COLORS
@color-main: #ffc75b; // Main color
@color-text-main: #333; // Main color fo text
@color-bg: #fff; // Main color for background
@color-bg-overlay: rgba(0,0,0,.75); // Overlay for background photos or video (optional)

// Fonts
@font-01: 'Montserrat'; // Font for headings, accents
@font-02: 'Open Sans'; // Regular text

// BACKGROUNDS
@background-static-image: '@{path-images}/backgrounds/static_image.jpg'; // Main background (optional)
@bg-logo: '@{path-images}/logos/logo_black.png'; // Logo for background (about section)

Styles

Location

Styles are located in the static/styles folder. Files are represented as CSS and LESS.

Styles description

The list of the style files:

  • main.less, main.css – main template styles,
  • responsive.less, responsive.css – responsive/adaptive styles,
  • variables.less – variables for main.less,
  • theme.less – template theme for main.less
  • main_*.css – converted template theme from main.less with selected theme.less.

Vendor styles

Template uses 3rd-party styles, their location is – static/vendor/*. The script/library connected to the template can have its own styles set. We highly recommend you to connect them from the specified path.

LESS

Template includes a set of LESS-files. They enable you to modify styles easily due to the language features. Find all the details about LESS here.

The template does not use LESS-files directly. To connect them you need to install js-script allowing to convert LESS to CSS on the go. Alternatively we recommend you to download a LESS-compiler from here (windows) or here (mac).

Scripts

Location and content

All the user scripts are located in static/scripts folder, external libraries (such as jquery, bootstrap, etc) are located in static/vendor.

Template scripts

List of the main script files:

  • static/scripts/clients-carousel.js – script for client carousel,
  • static/scripts/drawer.js,
  • static/scripts/drawer-tilt.js – scripts for using left drawer (use on of them),
  • static/scripts/elastic-fields.js – script for auto-height of contact form fields,
  • static/scripts/fancybox.js – settings for Fancybox 2,
  • static/scripts/feedback.js – script for feedback function,
  • static/scripts/fitvids.js – init for responsive video (fitvids.js),
  • static/scripts/form-effects.js – script for form input effects,
  • static/scripts/fullscreen.js – script for fullscreen welcome section,
  • static/scripts/maps.js – google maps,
  • static/scripts/menu.js – script main nav,
  • static/scripts/parallax.js,
  • static/scripts/parallax-topic.js – settings for parallax images
  • static/scripts/portfolio.js – script for portfolio,
  • static/scripts/scroll-to.js – smoothing scroll,
  • static/scripts/search.js – script for search input (only decorate),
  • static/scripts/vegas.js – settings for vegas,
  • static/scripts/videobg.js – script for background video,

Vendor scripts

Template uses vendor scripts (and their dependencies), their location is – static/vendor/*. It is recommended to connect them from the specified path.

List of some vendor scripts:

  • Bootstrap – Bootstrap is the most popular HTML, CSS, and JS framework for developing responsive, mobile first projects on the web.
  • Fancybox 2 – a tool that offers a nice and elegant way to add zooming functionality for images, html content and multi-media on your webpages,
  • fitvids.js – a lightweight, easy-to-use jQuery plugin for fluid width video embeds,
  • Font Awesome – Font Awesome gives you scalable vector icons that can instantly be customized,
  • jQuery – a fast, small, and feature-rich JavaScript library,
  • jquery.validation – plugin makes simple clientside form validation easy,
  • parallax.js – a dirt simple parallax scrolling effect inspired by Spotify.com and implemented as a jQuery plugin
  • Retinajs – an open source script that makes it easy to serve high-resolution images to devices with retina displays,
  • Vegas – a jQuery/Zepto plugin to add beautiful backgrounds and Slideshows to DOM elements,
Vide.js not correctly work with last version of jQuery. While the developers will fix it, we did it ourselves. You must use fixed version file fixed_jquery.vide.js or fixed_jquery.vide.min.js (minified version).
